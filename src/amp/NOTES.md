# Supported OS

Tested on Alpine, Debian and Ubuntu.

# Get Started

Add the feature to your devcontainer.json: 

```devcontainer.json
{
  "image": "mcr.microsoft.com/devcontainers/base:ubuntu",
  "features": {
    "ghcr.io/rdhambarage/vs-code-amp/amp": {}
  }
}
```

Start your devcontainer and run: `amp`
